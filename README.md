# Admin Vue Pro

## Project setup

```
yarn
```

### Fast-start、hot-reloads、mock-server for development

- one terminal

```
yarn dev
```

- another terminal

```
yarn mock
```

### Compiles and minifies for production

```
yarn build
```

### Deploy on branch gh-pages

```
yarn deploy
```

### Run your unit tests

```
yarn test:unit
```

### Run your end-to-end tests

```
yarn test:e2e
```

### Lints and fixes files

```
yarn lint
```
